# 🏋️ Pesquisa de Academia

Este projeto foi desenvolvido durante meus estudos de **HTML5** no **freeCodeCamp** como parte da certificação **Responsive Web Design**.

## 📖 Sobre o projeto

O objetivo foi criar um formulário de pesquisa utilizando apenas HTML, simulando uma pesquisa de satisfação para uma academia.

O formulário coleta informações dos usuários sobre seus objetivos, frequência de treino, preferências e sugestões de melhorias.

## 🚀 Tecnologias utilizadas

- HTML5

## 📋 Funcionalidades

- Campo para nome
- Campo para e-mail
- Campo para idade
- Lista de objetivos
- Botões de opção (radio)
- Lista de treinos favoritos
- Caixas de seleção (checkbox)
- Área para comentários
- Botão de envio


## 🎯 Objetivo

Praticar a criação de formulários utilizando:

- `<form>`
- `<label>`
- `<input>`
- `<select>`
- `<option>`
- `<textarea>`
- `<button>`
- Validação com `required`
- Tipos de entrada (`email`, `number`, `radio`, `checkbox`)
